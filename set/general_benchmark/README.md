# PhysisFold Benchmark — `set/`

The scientific dataset for the professional benchmark. **38 small monomeric
proteins** (N ≤ 107; the 3 largest — `1AFL` 248, `1BYI` 224, `1HEL` 129 — were
removed), each with a native ground-truth structure plus real
evidence of several kinds. Every artifact is **real or real-derived** — there are
no synthetic structures. Per-structure provenance lives in each
`<PDB_ID>/PROVENANCE.md` (auto-generated by `code_parts/make_provenance.py`).

## Per-structure folder contract

```
set/<PDB_ID>/
  sequence/          native PDB + FASTA          (ground-truth, RCSB PDB)
  homolog/           candidate homolog PDBs       + rcsb_sequence_cluster.json
  template_selection.json   record of which homolog was chosen as primary template
  msa/               <id>.a3m (ColabFold MSA) + coevolution_contacts.csv (MI+APC)
  experimental/      real wet-lab restraints, one subfolder per Input-branch
                     [built by code_parts/build_experimental.py]
    nmr_restraints/  <id>.str (wwPDB NMR-STAR) + <id>.nef (NEF, where remediated)
                     + raw/<id>.mr (original author restraints)   → --input-star / --input-nef
    noe_contacts/    <id>.csv  (CA-CA NOE, parsed from the .mr)    → --input-contacts
    ambiguous/       <id>.csv  (ambiguous NOE groups)             → --input-ambiguous
    disulfides/      <id>.csv  (SSBOND covalent locks, native)    → --input-disulfides
    provenance.json  per-structure experimental-source record
  af2/               real AlphaFold DB model (geometry-only)   [built by expand_inputs.py]
  esmfold/           real ESMFold v1 model (geometry-only)    [built by expand_inputs.py]
  conditions/        pre-built degraded distogram artifacts    [built by bench_builder.py]
    0pct_masked/   <id>.distogram.json   (L0 — full oracle distogram)
    50pct_masked/  <id>.distogram.json   (L1 — 50% long-range pairs masked)
    80pct_masked/  <id>.distogram.json   (L2 — 80% masked)
    noise_pm1/     <id>.distogram.json   (L3A — ±1 Å noise)
    noise_pm3/     <id>.distogram.json   (L3B — ±3 Å noise)
  PROVENANCE.md      truthful per-structure source record
```

> **`experimental/` is real-only.** NMR restraints are the **official wwPDB files**
> (remediated NMR-STAR / NEF / original `.mr`) for the SOLUTION-NMR entries;
> disulfides are extracted from the native **SSBOND** records (1-based sequential
> index, validated against the native CA-CA distance). There are **no synthetic
> restraints** — the earlier synthetic-crosslink scratch was deleted. The legacy
> `noise-spare/` folder was replaced by this tree.

`raw evidence` directories (`homolog/`, `msa/`, `experimental/`, etc.) are kept
**flat and as-is** — they are the source material and document how the set was
built. The `first_round` benchmark consumes the directories below.

The **selected primary template** for each structure lives at the set root in
`primary_templates/<query>_<template>.pdb` (e.g. `1VII_2F4K.pdb`), chosen by
`code_parts/build_set.py::select_primary_template` (leak-aware identity/coverage)
and recorded per structure in `template_selection.json`. The per-structure
`template/` copies were removed — `primary_templates/` is the single source.
4 structures have **no** primary template (homolog condition skips them):
`1HVZ`, `1R1F`, `1SN3`, `1YCC`.

## Condition → CLI input mapping (`first_round`)

The benchmark drives the **public generic CLI** exactly as a customer would
(`physis_fold_oracle/physis_fold_oracle_generic.py`). Each `first_round`
condition maps to one input flag:

| Condition | Source in `set/` | CLI input |
|---|---|---|
| `0pct_masked`  | `conditions/0pct_masked/<id>.distogram.json`  | `--distogram-artifact` |
| `50pct_masked` | `conditions/50pct_masked/<id>.distogram.json` | `--distogram-artifact` |
| `80pct_masked` | `conditions/80pct_masked/<id>.distogram.json` | `--distogram-artifact` |
| `noise_pm1`    | `conditions/noise_pm1/<id>.distogram.json`    | `--distogram-artifact` |
| `noise_pm3`    | `conditions/noise_pm3/<id>.distogram.json`    | `--distogram-artifact` |
| `80pct_masked_soi` | same as `80pct_masked` artifact | `--distogram-artifact` + `--soi-fill smart` |
| `homologs`     | `primary_templates/<id>_<tmpl>.pdb` | `--input-template` |
| `af2`          | `af2/<id>_af2.pdb`          | `--input-af2` |
| `esmfold`      | `esmfold/<id>_esmfold.pdb` | `--input-af2` (geometry-only) |
| `coevolution`  | `msa/coevolution_contacts.csv` | `--input-coevolution` |
| `nmr_star`     | `experimental/nmr_restraints/<id>.str` | `--input-star` |
| `nmr_nef`      | `experimental/nmr_restraints/<id>.nef` | `--input-nef` |
| `noe`          | `experimental/noe_contacts/<id>.csv`   | `--input-contacts` |
| `disulfides`   | `experimental/disulfides/<id>.csv`     | `--input-disulfides` |
| `ambiguous`    | `experimental/ambiguous/<id>.csv`      | `--input-ambiguous` |

> **Coverage (of 38):** `nmr_star` 14 · `nmr_nef` 3 · `noe` 13 · `disulfides` 20 ·
> `ambiguous` 1. Structures lacking a given evidence type are recorded as `skipped`
> in `manifests/index.json` for that condition.

All conditions use `--sequence sequence/<id>.pdb` as the coordinate-bearing
reference, and are scored against that native by `code_parts/scoring.py`.
First-round runs use **best-of-10** for transparency and maximum result.

## Data sources (real, documented)

| Asset | Source | Tool that built it |
|---|---|---|
| Native PDB + FASTA | RCSB PDB | `build_set.py` |
| Homologs + cluster | RCSB sequence cluster API | `build_set.py` |
| Primary template (`primary_templates/`) | leak-aware identity/coverage selection | `build_set.py::select_primary_template` |
| MSA (`.a3m`) | ColabFold (MMseqs2 servers) | `build_msa_colabfold.py` |
| Coevolution contacts | MI + APC over the MSA | `coevolution_apc.py` / `coev_to_contacts.py` |
| NMR/NOE restraints | experimental `.mr` (XPLOR/CNS) | `nmr_mr_to_contacts.py` |
| AF2 model | AlphaFold DB (EBI), by UniProt accession | `expand_inputs.py` |
| Masked/noise distograms | degradation of the native oracle distogram | `bench_builder.py` |

> **Real-only policy.** Masked/noise conditions are degradations of the distogram
> computed from the *native* coordinates — they perturb real evidence, they do not
> invent structure. AF2 models are genuine AlphaFold DB predictions (geometry-only;
> pLDDT/PAE are ignored by the engine). Any synthetic helper data produced by
> `expand_inputs.py` (e.g. DSS crosslinks) is **not** consumed by `first_round`.
